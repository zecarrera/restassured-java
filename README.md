# restassured-java

## Summary
This repo contains an example API testing framework using:
- RestAssured - API test library
- JUnit5 - Test runner
- Java
- Gradle
- GitHub actions

## Running tests 

### Locally

Running from the terminal:

```shell 
./gradlew build to build project
./gradlew test to run tests
```
